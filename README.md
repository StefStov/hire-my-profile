# hire-my-profile
Hire my profile is a project I started for me to learn to work with vue and show off my talents for interested parties. In this project I work with Vue and Sass. 
## Installation
Checkout the master branch

Do the following steps in the terminal.

Step 1
```bash
npm install
```
Step 2 for development
```bash
npm run serve
```
Step 2 for production
```bash
npm run build
```
Extra for lints and fixing files
```bash
npm run lint
```
